📋 Figma MCPサーバー セットアップ手順

ステップ 1: Figmaデスクトップアプリの設定

  1. Figmaデスクトップアプリを開く
  2. 左上の Figmaメニュー → 基本設定
  3. 「Dev Mode MCPサーバーを有効にする」 を選択
  4. 画面下部に確認メッセージが表示されることを確認
⚠️ 注意: Figmaデスクトップアプリからのみ利用可能です

ステップ 2: サーバー状態の確認

サーバーは以下のアドレスで動作します：

http://127.0.0.1:3845/sse

ステップ 3: Cursor での設定

以下の設定をCursor のMCP設定に追加してください：

{ "mcpServers": { "figma": { "url": "http://127.0.0.1:3845/sse" } } }

💡 使用例

Cursor での使用方法

  1. Figmaでフレームやレイヤーを選択
  2. Cursor のAIチャットで指示：
「選択中のFigmaフレームをReactコンポーネントとして実装してください」 「このデザインのCSSスタイルを生成してください」 「Figmaのデザインシステムからコンポーネントを抽出してください」

🆘 トラブルシューティング

接続エラーの場合

  • Figmaデスクトップアプリが起動していることを確認
  • Dev Mode MCPサーバーが有効になっていることを確認
  • ファイアウォールでポート3845がブロックされていないか確認
